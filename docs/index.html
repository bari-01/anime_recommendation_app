Getting Started

API Key Setup

AnimeRec uses the MyAnimeList API v2. Here's how to get your own API key:

1

Create a MAL Account

If you don't have one, sign up at myanimelist.net.

2

Register an API Application

Go to myanimelist.net/apiconfig and click "Create ID". Fill in your app name, description, and set the redirect URL to animerec://callback.

3

Copy Your Client ID

After registration, copy the Client ID from your API application page.

4

Update the App

Open AnimeRecApp.kt and replace the CLIENT_ID constant with your own:

const val CLIENT_ID = "your_client_id_here"

5

Build & Run

Connect an Android 12+ device or emulator and run ./gradlew assembleDebug to build your APK.
